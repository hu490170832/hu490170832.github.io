前置准备

  1. 开发环境

    Node 环境 （必须）
    VS Code 开发工具 （非必须，你可以使用你喜欢的开发工具）

  2. 注册 NPM 账号

    访问 NPM 官网 ，点击 Sign up 字样，输入邮箱用户名密码就可以完成创建了，注意这里的 Username 和 Password 将用于后续登录和发布包。Public Email 用于接收推送邮件。
